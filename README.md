I have created To-Do List App using HTML, CSS and JavaScript. 
In this i have added many task like Study, Watching TV, Outdoor Activities, Reading etc.
I have used hover effect on X button for better view.
Purpose of this app to know that which tasks we have to do today. 
We can remove task permanantly or erase the task after completion.
We can Add different tasks in list anytime/anyday according to our schedule.
